# Question 3 - Pattern Output

โจทย์ให้รับค่า 3 ตัว:

- ตัวอักษรที่จะแสดง
- จำนวนตัวอักษรมากสุด
- จำนวนรอบ

## Files

- `index.html`: หน้าจอดำสำหรับทดลองใน browser
- `pattern.js`: logic หลัก ใช้ได้ทั้ง browser และ Node.js
- `pattern-cli.js`: ตัวรันผ่าน command line
- `EXPLANATION.md`: คำอธิบาย logic

## Run

```bash
node pattern-cli.js 1 4 3
```
